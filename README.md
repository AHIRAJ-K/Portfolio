# Ahiraj K — Portfolio

Personal portfolio website showcasing my projects and skills in Electronics & Communication Engineering.

**Live site:** [Portfolio](https://ahiraj.xyz)

## Sections
- **Home** — Introduction and quick links
- **Projects** — Embedded systems and hardware projects
- **Skills** — Technical skills in hardware, software, and design
- **Experience** — Education and certifications
- **Contact** — Contact form and social links

## Tech Stack
- React, Vite, TypeScript, Tailwind CSS, shadcn-ui

## Run Locally
```bash
git clone https://github.com/AHIRAJ-K/portfolio.git
cd portfolio
npm install
npm run dev
```
